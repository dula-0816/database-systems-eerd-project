# Database Systems – EERD Design Project

This project demonstrates conceptual and logical database design 
for two business scenarios: Foodz@Yourz and BoilHeaterz.

## Topics Covered
- Enhanced Entity Relationship Diagrams (EERD)
- UML-based database modelling
- Entity specialization
- Relationship multiplicities
- Logical ERD mapping
- Database design decision logs

## Tools Used
- UML modelling
- Database design principles
